# Q1：Dify 如果設定好了之後，可以把我的工作流直接給其他人複製貼上使用嗎？

是的！Dify 支援工作流程的分享和複製，有幾種方式：

## 🔄 工作流程分享方式

### 1. **DSL 檔案匯出/匯入**
- ✅ **匯出 DSL（YAML/JSON）檔案**
  - 包含完整工作流程設定
  - 節點配置、連接關係都會保留
  - 其他人可直接匯入使用

### 2. **應用程式發布分享**
- **發布為 Web App**：生成連結直接分享使用
- **API 分享**：提供 API endpoint 給其他人調用
- **嵌入網頁**：透過 iframe 嵌入碼分享

## ⚠️ 注意事項

### 需要調整的部分：
```
❌ 不會複製的內容：
- API 金鑰（OpenAI、Claude 等）
- 知識庫檔案（需重新上傳）
- 外部工具連接設定
- 環境變數

✅ 會保留的內容：
- 工作流程邏輯
- Prompt 設定
- 節點配置
- 變數設定
```

## 🎯 實用技巧

**建立模板庫**：
- 將常用工作流程做成標準模板
- 團隊內部可快速複製使用
- 定期更新優化版本

**版本管理**：
- 用 Git 管理 DSL 檔案
- 記錄每次更新內容
- 方便團隊協作
